1. 利用接口做参数，写个计算器，能完成加减乘除运算。 

（1）定义一个接口 Compute 含有一个方法 int computer(int n, int m)。 

（2）设计四个类分别实现此接口，完成加减乘除运算。 

（3）设计一个类 UseCompute，类中含有方法：public void useCom(Compute com, 

int one, int two)， 

此方法能够用传递过来的对象调用 computer 方法完成运算，并输出运算的 

结果。 

（4）设计一个主类 Test，调用 UseCompute 中的方法 useCom 来完成加减乘除运算。 

```java
// 定义接口 Compute
interface Compute {
    int computer(int n, int m);
}

// 实现加法类
class Add implements Compute {
    public int computer(int n, int m) {
        return n + m;
    }
}

// 实现减法类
class Subtract implements Compute {
    public int computer(int n, int m) {
        return n - m;
    }
}

// 实现乘法类
class Multiply implements Compute {
    public int computer(int n, int m) {
        return n * m;
    }
}

// 实现除法类
class Divide implements Compute {
    public int computer(int n, int m) {
        return n / m;
    }
}

// 使用计算器类
class UseCompute {
    public void useCom(Compute com, int one, int two) {
        int result = com.computer(one, two);
        System.out.println("运算结果: " + result);
    }
}
public class Test {
    public static void main(String[] args) {
        UseCompute uc = new UseCompute();
        int num1 = 10;
        int num2 = 5;

        Compute add = new Add();
        uc.useCom(add, num1, num2); // 调用加法运算

        Compute subtract = new Subtract();
        uc.useCom(subtract, num1, num2); // 调用减法运算

        Compute multiply = new Multiply();
        uc.useCom(multiply, num1, num2); // 调用乘法运算

        Compute divide = new Divide();
        uc.useCom(divide, num1, num2); // 调用除法运算
    }
}
```

2.编写程序接收用户输入分数信息，如果分数在 0—100 之间，输出成绩。如果成 

绩不在该范围内，抛出异常信息，提示分数必须在 0—100 之间。 

```java
import java.util.Scanner;

public class GradeChecker {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("请输入分数: ");
        int score = scanner.nextInt();

        try {
            checkGrade(score);
            System.out.println("成绩: " + score);
        } catch (IllegalArgumentException e) {
            System.out.println("异常信息: " + e.getMessage());
        }
    }

    public static void checkGrade(int score) {
        if (score < 0 || score > 100) {
            throw new IllegalArgumentException("分数必须在0-100之间");
        }
    }
}
```

3.编写一个计算 N 个整数平均值的程序。程序提示用户输入，必须输入所有 N 个数。如果用户输入的值是一个负数，则应该抛出一个异常并捕获， 

提示“必须是正数或者0”。并提示用户再次输入该数。 

```java
import java.util.Scanner;

public class AverageCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("请输入整数的个数 N: ");
        int n = scanner.nextInt();

        int sum = 0;
        int count = 0;

        for (int i = 1; i <= n; i++) {
            System.out.print("请输入第 " + i + " 个整数: ");
            int num = readPositiveNumber(scanner);

            sum += num;
            count++;
        }

        double average = (double) sum / count;
        System.out.println("平均值: " + average);
    }

    public static int readPositiveNumber(Scanner scanner) {
        while (true) {
            int num = scanner.nextInt();
            if (num >= 0) {
                return num;
            } else {
                System.out.println("必须是正数或者0，请重新输入: ");
            }
        }
    }
}
```

4.定义一个 Employee 类，该类包含： private 成员变量 name,number,birthday， 

其中 birthday 为 MyDate 类的对象； abstract方法 earnings()； toString() 方法输出对象的 name,number 和 birthday。 

```java
import java.util.Date;

abstract class Employee {
    private String name;
    private String number;
    private MyDate birthday;

    public Employee(String name, String number, MyDate birthday) {
        this.name = name;
        this.number = number;
        this.birthday = birthday;
    }

    public abstract double earnings();

    @Override
    public String toString() {
        return "姓名：" + name + "，工号：" + number + "，生日：" + birthday.toString();
    }
}

class MyDate {
    private int year;
    private int month;
    private int day;

    public MyDate(int year, int month, int day) {
        this.year = year;
        this.month = month;
        this.day = day;
    }

    @Override
    public String toString() {
        return year + "年" + month + "月" + day + "日";
    }
}

// 示例子类
class Manager extends Employee {
    private double salary;

    public Manager(String name, String number, MyDate birthday, double salary) {
        super(name, number, birthday);
        this.salary = salary;
    }

    @Override
    public double earnings() {
        return salary;
    }
}

class EmployeeTest {
    public static void main(String[] args) {
        MyDate birthday = new MyDate(1990, 5, 15);
        Employee employee = new Manager("John", "001", birthday, 5000.0);
        System.out.println(employee.toString());
        System.out.println("收入：" + employee.earnings());
    }
}

```

5.给定字符串 S 和单词字典 words, 求 words[i] 中是 S 的子序列的单词个数。 

示例: 

输入: 

S = “abcde” 

words = [“a”, “bb”, “acd”, “ace”] 

输出: 3 

解释: 有三个是 S 的子序列的单词: “a”, “acd”, “ace”

```java
import java.util.ArrayList;
import java.util.List;

class SubsequenceWords {
    public static int countSubsequenceWords(String S, String[] words) {
        int count = 0;

        for (String word : words) {
            if (isSubsequence(word, S)) {
                count++;
            }
        }

        return count;
    }

    private static boolean isSubsequence(String word, String S) {
        int i = 0;
        int j = 0;

        while (i < word.length() && j < S.length()) {
            if (word.charAt(i) == S.charAt(j)) {
                i++;
            }

            j++;
        }

        return i == word.length();
    }

    public static void main(String[] args) {
        String S = "abcde";
        String[] words = {"a", "bb", "acd", "ace"};

        int count = countSubsequenceWords(S, words);
        System.out.println("输出: " + count);
    }
}
```
